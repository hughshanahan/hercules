This folder contains result data for Phase I of the Hercules transcriptomics analysis system.
This comprises of k-mer motif results, where k=4 (fourmers), for two Drosophila Melanogaster datasets.

Each sub-folder contains the 4-mer results which are stored in herc-final-*.csv, where * is the 4-mer,
for instance AAAA. Data format is exon_key, position, count.
